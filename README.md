# JFrameVendas.java 🛍️ 
Este projeto é uma aplicação Java com interface gráfica utilizando a biblioteca Swing (JFrame). Destina-se ao gerenciamento de vendas, incluindo cadastro de produtos, clientes e controle de estoque.

## Funcionalidades 📋 
Cadastro de Produtos: Adicione, edite e remova produtos do estoque.
Cadastro de Clientes: Registre informações dos clientes para facilitar o processo de venda.
Controle de Estoque: Monitore a quantidade de produtos disponíveis.
Registro de Vendas: Efetue vendas e mantenha um histórico.
Interface Gráfica: Utiliza JFrame para uma interface amigável.​

## Tecnologias Utilizadas 🛠️ 
Linguagem: Java
Biblioteca Gráfica: Swing (JFrame)
IDE Recomendada: NetBeans

## Como Executar 🚀 
Clone este repositório em sua máquina local:​
bash
git clone https://github.com/kenzoScript/JFrameVendas.java.git

Abra o projeto na IDE NetBeans.
Compile e execute a classe principal Form.java.
A interface gráfica será carregada, permitindo o uso do sistema de vendas.
